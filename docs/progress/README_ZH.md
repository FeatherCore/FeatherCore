# FeatherCore 计划与进度文档

这个目录专门记录当前开发计划、真实进度、下一步任务和 milestone 状态。

这里的文档要遵守一个原则：

```text
计划可以前瞻
进度必须对应当前代码真实状态
```

## 当前文档

```text
GRAPHICS_ROADMAP_AND_PROGRESS_ZH.md
  Feather 图形路线总计划与进度记录，包括 FRender、WING GUI、WING Desktop、Pinion、NuttX graphics 的关系和下一步任务。

WING_GUI_PROGRESS_AND_UPDATE_PLAN_ZH.md
  WING GUI 当前真实进度、近期更新、下一步计划和验证方式。

WING_GUI_IMPLEMENTATION_PROGRESS_AND_PLAN_ZH.md
  WING GUI 具体实现进度、组件能力、runtime 能力和后续实现拆解。
```

## 使用方式

设计文档回答：

```text
为什么这样做
架构边界是什么
长期方向是什么
```

本目录回答：

```text

```
