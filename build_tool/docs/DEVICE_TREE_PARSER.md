# FeatherCore Build Tool - Device Tree Parser

This module parses device tree source files and generates Rust code for the `common/generated` library.

## Architecture

```
build_tool/ (std environment)
    ↓ parses
platform/board/xxx/board.dts
    ↓ generates
common/generated/src/generated/devicetree.rs (no_std)
    ↓ used by
boot/, kernel/, rootfs/
```

## Usage

```rust
use device_tree_parser::DeviceTreeParser;

// Parse DTS file
let node = DeviceTreeParser::parse_file("board.dts")?;

// Generate Rust code
let code = DeviceTreeParser::generate_rust_code(&node);

// Write to common/generated/src/generated/devicetree.rs
std::fs::write("common/generated/src/generated/devicetree.rs", code)?;
```

## Generated Code Example

```rust
//! Generated by feathercore-build from device tree source

    pub const FLASH_SIZE: u32 = 0x200000;
    pub const RAM_BASE: u32 = 0x20000000;
    pub const RAM_SIZE: u32 = 0x30000;
}

pub mod peripherals {
    pub const COUNT: usize = 5;
}
```
